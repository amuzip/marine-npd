# Research Workflow
## Pipeline
1. Compound identification from marine species
2. Structure file processing (SDF/MOL)
3. PASS biological activity prediction
4. PharmaExpert drug-likeness assessment
5. Target-disease mapping
6. Dashboard generation
7. Report compilation
## Tools
- Python: RDKit, pandas, plotly
- PASS Online / PharmaExpert
- Obsidian: Knowledge management (PARA + Zettelkasten)
